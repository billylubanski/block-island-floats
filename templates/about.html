The short version

What this app is tracking

Each year glass artist Eben Horton hides numbered glass floats around Block Island. This tracker turns the public reporting archive into a practical read on where hunters have historically succeeded.

How the tracker helps

The app combines archived reports, location normalization, seasonal analysis, a lightweight forecast model, and live weather context so you can choose where to start with more discipline than guesswork alone.

What it is not

This is not an official hiding map and it does not guarantee a find. It is a field-read tool built from public reporting patterns, which means uncertainty and incomplete records are part of the dataset.

Field use

Best used as a decision aid

Open the dashboard to read the archive, switch to field mode once you are moving, and use location detail when one trail or beach starts to matter more than the rest.

Respect the island

Locations are approximate, many reports are incomplete, and trail etiquette still matters more than the model. Stay on established paths, avoid dunes and restricted places, and treat the tracker as guidance rather than permission.

Core surfaces

How the experience is organized

Each part of the app has a specific job so you can move from overview to field action without losing the thread.

Dashboard

Map-led archive view for recovery patterns, strongest starting zones, and year-by-year context.

Field mode

Mobile-first operational view for geolocation, map orientation, and quick navigation links.

Forecast

Compact briefing that combines seasonality, ranked locations, and weather context.

Location detail

Place-specific story with images, quick-read stats, and the deeper chronology when you need it.

Search

Fast archive lookup by float number, finder, or place name before you jump into detail.

PWA support

Installable on mobile so the app is easier to reach when you are already on the island.

Install

Use it like a field shortcut

Installing the app keeps the key surfaces one tap away and improves the mobile flow once you are moving between spots.

  • iPhone / Safari: open the share sheet and choose Add to Home Screen.
  • Android / Chrome: use the browser menu and choose Install app or Add to Home Screen.
