About the Project

The Block Island Glass Float Tracker is a comprehensive hunting assistant that combines data analysis, machine learning, and real-time weather integration to help float hunters improve their chances of finding a glass float on Block Island.

What are Glass Floats?

Every year, glass artist Eben Horton hides hundreds of numbered glass floats (orbs) along the trails and beaches of Block Island. They are dated and numbered. If you find one, it's yours to keep!

How This Works

This application combines web scraping, data analysis, machine learning, and real-time weather integration to build a comprehensive hunting assistant. We analyze the official registry of found floats to:

  • Identify "hot spots" where floats are frequently found.
  • Analyze seasonal trends and predict optimal hunting times using machine learning.
  • Visualize recovery rates (found vs. still hidden) for each year.
  • Track real-time weather conditions for field hunters.
  • Provide GPS-based navigation to nearby hunting grounds.
  • Display photo galleries of found floats at each location.

Key Features

  • Interactive Dashboard: Heatmap visualization with year filtering and recovery rate analysis
  • Field Mode: Mobile-first interface with GPS, real-time weather, and navigation links
  • ML Forecast: Daily predictions based on historical patterns and seasonality scoring
  • Location Details: Photo galleries, statistics, and complete find history for each spot
  • Advanced Search: Full-text search across finders, locations, and float numbers
  • PWA Support: Install on your device for offline access and native app experience
